ClawControl supports multiple cloud providers to give you flexibility in deploying OpenClaw instances. Each provider offers different pricing, regions, and server configurations.

## Pricing Comparison

Based on the default configurations used by ClawControl:

| Provider | Default Plan | vCPUs | RAM | Storage | Price/Month |
|----------|--------------|-------|-----|---------|-------------|
| Hetzner | CPX11 (US East) | 2 | 2 GB | 40 GB | ~$4.99 |
| DigitalOcean | s-1vcpu-2gb (NYC1) | 1 | 2 GB | 50 GB | $12.00 |

<Tip>
  Hetzner offers the most cost-effective option for basic deployments, while DigitalOcean provides broader global coverage.
</Tip>

## Provider Selection

When choosing a provider, consider:

- **Budget**: Hetzner offers lower pricing for similar specifications
- **Region**: Choose a provider with servers close to your users
- **Compliance**: Consider data residency requirements for your use case
- **Experience**: Use a provider you're already familiar with

## Getting Started

To deploy with a specific provider:

1. Get an API key from your chosen provider:
   - [Hetzner API Key](/providers/hetzner#api-key-setup)
   - [DigitalOcean API Token](/providers/digitalocean#api-token-setup)

2. Run the deployment wizard:
   ```bash
   clawcontrol /new
   ```

3. Select your provider and configure your deployment

4. Deploy to the cloud:
   ```bash
   clawcontrol /deploy
   ```

## Provider Configuration

Each provider has specific configuration options that can be customized:

### Hetzner
- **Server Type**: Choose from CPX, CX, or CCX series
- **Location**: Available in US East (ash), EU regions
- **Image**: Ubuntu 24.04 (default)

### DigitalOcean
- **Size**: Choose from various droplet sizes
- **Region**: NYC, SFO, EU, Asia-Pacific regions
- **Image**: Ubuntu 24.04 (default)
